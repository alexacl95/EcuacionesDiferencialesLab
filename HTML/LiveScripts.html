Live Scripts en Matlab

¿Qué es un Live Script?

Los Live Scripts son documentos interactivos que combinan el código de MATLAB con texto, ecuaciones e imágenes en un solo entorno llamado Live Editor. Adicionalmente, estos Live scripts almacenan y muestran la salida junto con el código que la crea (para mayo información visitar [1]).
  1. Para crear Live Scrips con Live Editors diríjase a la tabla de Home y seleccione la opción de New Live Script.
  2. Al crear el Live script aparecerá en la barra la opción de Live editor e Insert, las cuales presentan las opciones de edición de texto e imágenes.
En el archivo de Live Script se puede:
  1. Agregar código
  2. Correr código
  3. Mostrar la salida del código
  4. Dar formato a texto
  5. Agregar imágenes e hipervínculos
Para mayor información ver [2]
